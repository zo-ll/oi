# oi

Tiny Go agent runtime.

## Status

Scaffolded rebuild in progress.

Current focus:
- OpenAI-compatible providers
- safe local tools
- stdio RPC for bot integration
- standard library only

## Commands

```bash
oi doctor
oi models
oi run "task"
oi chat
oi rpc
```

## Current status

Available now:

```bash
oi doctor
oi models
```

`chat`, `run`, and `rpc` are scaffolded but not implemented yet.
